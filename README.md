# Information Search

**Выполнил**: Ермаков Ярослав Валерьевич  
**Группа**: М80-407Б-22

Этот проект предназначен для создания системы информационного поиска с использованием таких техник обработки текста, как токенизация, стемминг, булев поиск и анализ частот терминов. Он реализует основные этапы обработки текстов документов и построения индекса для быстрого поиска.

## Команды для работы

**Примечание**

### 1. Установка зависимостей

Для установки всех зависимостей проекта выполните:

```bash
make install
```

### 2. Скачивание документов (поисковый робот)

Для скачивания документов в MongoDB используйте команду:

```bash
make download CFG=path/to/config.yaml
```

Для отслеживания статистики скачивания есть утилита:

```bash
make monitor CFG=path/to/config.yaml
```

### 3. Токенизация

Токенизация текстов документов и экспорт статистики:

```bash
make tokenize STEMMING=1
```

### 4. Анализ частот и построение графика Ципфа

Для анализа частот терминов и построения графика распределения по закону Ципфа:

```bash
make zipf
```

### 5. Построение булевого индекса

Для построения булевого индекса по документам:

```bash
make index
```

### 6. Булев поиск

Для выполнения булевого поиска по индексу:

```bash
make search Q='(bert | transformer) & !survey'
```

### 7. Полный пайплайн

Для выполнения полного пайплайна (от скачивания до индексации):

```bash
make full
```

### 8. Очистка результатов

Для очистки сгенерированных файлов и установленных зависимостей:

```bash
make clean
```
